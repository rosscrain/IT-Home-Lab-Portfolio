# IT Home Lab Portfolio

This repository documents a series of home labs built to develop and demonstrate IT help desk skills.

Each lab includes step by step documentation, screenshots, and honest notes on what worked and what did not.

## Labs

- Lab 1: RDP Remote Access
- Lab 2: Spiceworks Cloud Help Desk Ticketing System
- Lab 3: Active Directory User Management
- Lab 4: Network Troubleshooting

## Environment

- Host: Windows 11
- VM: Windows 10 Pro in VirtualBox 7.2.6
- Secondary VM: Windows Server 2022 configured as a domain controller (lab.local)
- RAM: 16GB
- Storage: 2TB SSD

## About

IT home lab portfolio documenting hands-on help desk skills including remote access, ticketing, Active Directory administration, and network troubleshooting. Built on Windows 11 with VirtualBox, Windows 10 Pro, and Windows Server 2022.
